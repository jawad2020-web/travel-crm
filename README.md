# InSync Ecommerce CRM

## Environments

```
1. Production

==> Branch Name: main

==> Domain Name: main.insync-crm.com

2. Development

==> Branch Name: develop

==> Domain Name: develop.insync-crm.com
```

## Install

```
git clone https://github.com/abendsoft/insync-crm.git
cd insync-crm
yarn install
```

## Start development

```
yarn dev
```

## Build for production

```
yarn build
```

## Run build for production

```
yarn start
```
